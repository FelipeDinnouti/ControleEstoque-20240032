## API Rest de Controle de Estoque em Java

Esse é um projeto simples para demonstrar o uso de Spring Boot em Java, utilizando Hibernate/JPA e Lombok para gerenciar um estoque ficicio de produtos, realizando cadastro de produtos e vendas.

### Rodar o projeto

Antes de iniciar o projeto, verifique a sua instalação de Java, utilizando a versão 17.

E então, mais nenhuma dependência é necessária e se roda o projeto com o seguinte comando:

(Linux)
```bash
./mvnw spring-boot:run
```
